# PEDIT-P releases

Public, immutable pilot release assets for 현장 작업계획서 통합관리 (`workplan-studio`).

Each release contains exactly the Windows ZIP, `app-manifest.json`, `app-manifest.jws`, and `SHA256SUMS.txt`.

`v0.1.1-pilot.1` and `v0.1.1-pilot.2` are superseded and must not be installed. Use `v0.1.1-pilot.3` only.
